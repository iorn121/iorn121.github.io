Running

Overview

心身のリフレッシュと、目標に向かって着実にステップアップするプロセスを楽しむために走っています。
普段は週2〜3回、月間約50kmのペースを維持。平日の夜ランを中心に、大会前には距離を伸ばしてコンディションを整えています。

Home Courses

  • 代々木公園: 緑豊かな環境で季節を感じながらのジョギングに。
  • 皇居周辺: 言わずと知れた定番コース。
  • 神宮球場周辺: 一定のペースで走り込みたい時の定番コース。
